# fourier_explorations

Computational explorations of Fourier analysis concepts, connecting electrical engineering intuition to astronomical data analysis methods.

Jupyter notebooks building intuition for:
- Fourier transform properties and pairs
- Dirac delta functions and the Dirac comb
- Sampling theory and aliasing
- The Lomb-Scargle periodogram (VanderPlas 2018)

Inspired by working through VanderPlas (2018), ApJS 236, 16.
